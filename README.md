# Реализация парсеров: Эрли и LR(1)

Данный репозиторий содержит практическую реализацию двух классических алгоритмов синтаксического анализа и тестов для них в рамках курса "Формальные языки и трансляции":
1.  **Алгоритм Эрли (Earley Parser)** — восходящий парсер, работающий с произвольными КС-грамматиками.
2.  **LR(1)-парсер** — восходящий парсер для детерминированных грамматик, построенный на основе LR(1)-таблиц.

После прохождения ревью за задачи получен максимальный балл.
