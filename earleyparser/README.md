# Earley Parser


## Сборка и запуск

###  Сборка

#### Для запуска парсера:

```bash
g++ -std=c++23 earley.cpp -o earley.out
./earley.out
```

Программа ожидает ввод грамматики и список слов. Формат ввода соответствует условию тестов.

### Пример входных данных

```
1 2 2
S
( )
S ->
S -> (S)S
S
4


()
(())
(()
```

## Ожидаемый вывод:
```
YES
YES
NO
YES
```

#### Для запуска тестов:

```bash
g++ -std=c++23 tests.cpp -o tests.out
./tests.out
```

Если всё работает, вы увидите:

```
[OK] TestName
...
Тесты успешно пройдены. (N/N)
```

##  Структура проекта

* `earley.cpp` — реализация алгоритма Эрли
* `tests.cpp` — тестовый фреймворк
* `README.md` — инструкция по запуску
